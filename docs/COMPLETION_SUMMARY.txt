🚀 HOW TO RUN:

1. VERIFY MONGODB
   mongod  # Must be running on localhost:27017

2. SEED DATABASE (First time only)
   cd server
   node seed-complete.js

3. START BACKEND
   npm run dev
   # Server runs on: http://localhost:5000

4. START FRONTEND (From root directory)
   npm start
   # App opens on: http://localhost:3000
